# YPP-Gizmo
 
A Java desktop application for extracting and analysing crew performance data from the Yohoho! Puzzle Pirates (YPP) game client.
 
Previously maintained as a private repository; published here as a portfolio reference.
 
---
 
## Overview
 
YPP-Gizmo is a data extraction and analysis tool built with Java. It demonstrates practical application of GUI automation, real-time data processing, and multi-layered software architecture against a complex, third-party interface.
 
---
 
## Key Capabilities
 
- **Real-Time Data Extraction** — Processes live performance metrics directly from the game client UI
- **Multi-Station Analysis** — Handles data from multiple job types, each with distinct scoring algorithms
- **Persistent Storage** — Manages historical data and user preferences across sessions
- **Cross-Platform Compatibility** — Supports multiple game client implementations
- **Interactive GUI** — Interface for data visualisation, filtering, and export
---
 
## Technology Stack
 
- **Language:** Java
- **UI Framework:** Java Swing
- **Architecture Patterns:**
  - Event-driven programming
  - Component-based architecture
  - Reflection and introspection
  - Agent-based instrumentation
---
 
## Technical Highlights
 
Notable implementation challenges include:
 
-  Advanced Java reflection and runtime introspection
-  Complex GUI component analysis and data extraction from a live third-party application
-  Java instrumentation and agent development
-  Recursive algorithms for traversing hierarchical UI data structures
-  Multi-threaded event handling in Swing applications
